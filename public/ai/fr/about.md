# LLMO Framework

> Le guide définitif de l'optimisation pour les LLM — rendre votre contenu détectable par l'IA.

## Qu'est-ce que le LLMO ?

Le LLMO (Large Language Model Optimization) est la pratique d'optimisation du contenu web pour que les systèmes d'IA — ChatGPT, Claude, Gemini et Perplexity — puissent le découvrir, le comprendre et le citer avec précision.

Le LLMO est le framework général qui englobe l'AEO (Answer Engine Optimization) et le GEO (Generative Engine Optimization), tout en s'étendant au-delà de la recherche pour couvrir toutes les interactions avec les LLM, y compris les requêtes directes, le RAG et les agents IA.

## Pourquoi le LLMO est important

- L'usage des moteurs de recherche traditionnels diminuera de 25 % d'ici 2026 (Gartner, février 2024)
- Le trafic de recherche IA convertit 25× plus que la recherche traditionnelle (Go Fish Digital)
- L'ajout de statistiques améliore le taux de citation de +115,1 % (papier GEO, KDD 2024)
- Les mentions web sont 3× plus prédictives pour la visibilité IA que les backlinks (Ahrefs, 75 000 marques)

## Le framework LLMO (6 composants)

1. **Clarté des connaissances** — Contenu clair, factuel et sans ambiguïté
2. **Formatage structuré** — Markdown, JSON-LD, HTML sémantique, llms.txt
3. **Signaux de recherche** — llms.txt, répertoire /ai/, robots.txt, sitemap
4. **Signaux d'autorité** — Cohérence multi-plateforme, publications, expertise vérifiable
5. **Signaux de citation** — Sources primaires, statistiques, dates, références
6. **Signaux de cohérence** — Le même fait raconte la même histoire en HTML, JSON-LD, Markdown et llms.txt (source unique de vérité)

Chaque composant noté de 0 à 3. Score maximum : 18 points.
