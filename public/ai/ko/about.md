# LLMO 프레임워크

> LLM 최적화(LLM Optimization)를 위한 결정적 가이드 — AI가 콘텐츠를 발견할 수 있게 만들기.

## LLMO란?

LLMO(Large Language Model Optimization)는 ChatGPT, Claude, Gemini, Perplexity 같은 대형 언어 모델이 웹 콘텐츠를 정확히 발견·이해·인용할 수 있도록 최적화하는 실천법입니다.

LLMO는 AEO(Answer Engine Optimization)와 GEO(Generative Engine Optimization)를 포괄하는 상위 프레임워크로, 검색을 넘어 RAG와 AI 에이전트 등 모든 LLM 상호작용을 다룹니다.

## 왜 LLMO가 중요한가

- 전통적 검색 사용량은 2026년까지 25% 감소 예상(Gartner, 2024년 2월)
- AI 검색 트래픽은 전통 검색 대비 25배 높은 전환율(Go Fish Digital)
- 통계 추가로 인용률 +115.1% 향상(GEO 논문, KDD 2024)
- 웹 언급은 AI 가시성 예측에서 백링크보다 3배 효과적(Ahrefs, 75,000 브랜드)

## LLMO 프레임워크 (5개 컴포넌트)

1. **지식 명확성(Knowledge Clarity)** — AI가 정확히 이해·요약할 수 있는 명료한 콘텐츠
2. **구조화 포맷(Structural Formatting)** — Markdown, JSON-LD, 시맨틱 HTML, llms.txt
3. **검색 신호(Retrieval Signals)** — llms.txt, /ai/ 디렉토리, robots.txt, sitemap
4. **권위 신호(Authority Signals)** — 크로스 플랫폼 일관성, 출판물, 검증 가능한 전문성
5. **인용 신호(Citation Signals)** — 1차 출처, 통계, 날짜, 참고 문헌

각 컴포넌트 0-3점, 최고 15점.
