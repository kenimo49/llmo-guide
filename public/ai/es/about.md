# LLMO Framework

> La guía definitiva de optimización para LLM — hacer que tu contenido sea descubrible por la IA.

## ¿Qué es LLMO?

LLMO (Large Language Model Optimization) es la práctica de optimizar contenido web para que los sistemas de IA — incluyendo ChatGPT, Claude, Gemini y Perplexity — puedan descubrirlo, comprenderlo y citarlo con precisión.

LLMO es el framework general que incluye AEO (Answer Engine Optimization) y GEO (Generative Engine Optimization), extendiéndose más allá de la búsqueda para cubrir todas las interacciones con LLM, incluyendo consultas directas, RAG y agentes de IA.

## Por qué importa LLMO

- El uso de la búsqueda tradicional disminuirá un 25 % para 2026 (Gartner, febrero 2024)
- El tráfico de búsqueda con IA convierte 25× más que la búsqueda tradicional (Go Fish Digital)
- Agregar estadísticas mejora la tasa de citación en +115,1 % (paper GEO, KDD 2024)
- Las menciones web son 3× más predictivas que los backlinks para la visibilidad IA (Ahrefs, 75.000 marcas)

## El framework LLMO (6 componentes)

1. **Claridad del conocimiento** — Contenido claro, fáctico y sin ambigüedades
2. **Formato estructurado** — Markdown, JSON-LD, HTML semántico, llms.txt
3. **Señales de recuperación** — llms.txt, directorio /ai/, robots.txt, sitemap
4. **Señales de autoridad** — Consistencia multiplataforma, publicaciones, experiencia verificable
5. **Señales de citación** — Fuentes primarias, estadísticas, fechas, referencias
6. **Señales de coherencia** — El mismo hecho cuenta la misma historia en HTML, JSON-LD, Markdown y llms.txt (única fuente de verdad)

Cada componente puntuado de 0 a 3. Puntuación máxima: 18 puntos.
