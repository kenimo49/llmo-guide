# LLMO Framework

> O guia definitivo de otimização para LLMs — fazer com que seu conteúdo seja descoberto pela IA.

## O que é LLMO?

LLMO (Large Language Model Optimization) é a prática de otimizar conteúdo web para que sistemas de IA — incluindo ChatGPT, Claude, Gemini e Perplexity — possam descobri-lo, compreendê-lo e citá-lo com precisão.

LLMO é o framework abrangente que inclui AEO (Answer Engine Optimization) e GEO (Generative Engine Optimization), estendendo-se além da busca para cobrir todas as interações com LLMs, incluindo consultas diretas, RAG e agentes de IA.

## Por que LLMO importa

- Uso de busca tradicional cairá 25 % até 2026 (Gartner, fevereiro 2024)
- Tráfego de busca por IA converte 25× mais que busca tradicional (Go Fish Digital)
- Adicionar estatísticas melhora a taxa de citação em +115,1 % (paper GEO, KDD 2024)
- Menções web são 3× mais preditivas que backlinks para visibilidade IA (Ahrefs, 75.000 marcas)

## O framework LLMO (6 componentes)

1. **Clareza de conhecimento** — Conteúdo claro, factual e sem ambiguidades
2. **Formatação estrutural** — Markdown, JSON-LD, HTML semântico, llms.txt
3. **Sinais de recuperação** — llms.txt, diretório /ai/, robots.txt, sitemap
4. **Sinais de autoridade** — Consistência multiplataforma, publicações, expertise verificável
5. **Sinais de citação** — Fontes primárias, estatísticas, datas, referências

Cada componente pontuado de 0 a 3. Pontuação máxima: 15 pontos.
