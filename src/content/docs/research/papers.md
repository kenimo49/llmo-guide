## Core Papers

### GEO: Generative Engine Optimization
- **Authors**: Aggarwal, Murahari, Rajpurohit, Kalyan, Narasimhan, Deshpande
- **Venue**: KDD 2024 (ACM SIGKDD)
- **Link**: [arXiv:2311.09735](https://arxiv.org/abs/2311.09735)
- **Summary**: First academic framework for optimizing content visibility in generative search engines.

### llms.txt Proposal
- **Author**: Jeremy Howard
- **Link**: [llmstxt.org](https://llmstxt.org/)
- **Summary**: A proposal for a standardized file that provides information to LLMs about a website.
